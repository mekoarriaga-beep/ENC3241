# Proposal

Developing a Machine Learning Model to Predict Crime Hotspots in Urban Areas
Meko Arriaga							Computer Science 

Background 

Violent crime continues to be a major public safety concern in the United States, with significant variation across regions and communities. According to the Federal Bureau of Investigation (2023), “millions of violent crimes—including homicide, aggravated assault, robbery, and rape—are reported annually through the Uniform Crime Reporting (UCR) Program”. Research consistently shows that crime is not randomly distributed but instead clusters in hotspots, often influenced by environmental and socioeconomic conditions. Research supports the use of data-driven approaches to understanding and predicting crime patterns. For example, Gerber (2014) demonstrated that “social media and geospatial data can be used to improve crime prediction accuracy”. Additionally, studies in criminology (Sampson & Groves, 1989) have “linked crime rates to structural factors such as poverty, unemployment, and community organization,” reinforcing the importance of combining multiple data sources. There is a growing need for systems that are not only accurate but also understandable. As a computer science student pursuing a career in technology, this project aims to apply machine learning techniques to real world data while also emphasizing transparency. 

Methodology 

This study will use publicly available datasets to develop and evaluate a machine learning model for predicting crime hotspots. Crime data will be collected from the Federal Bureau of Investigation’s Uniform Crime Reporting (UCR) Program, while demographic and socioeconomic data will be obtained from the U.S. Census Bureau’s American Community Survey. The collected datasets will be processed using Python, specifically with libraries such as Pandas and NumPy. Cloud computing resources will be utilized through Google Cloud Platform, where data processing, model training, and deployment will be conducted using tools such as Google Colab and Cloud Run. This process will include handling missing values, removing inconsistencies, and standardizing variables across datasets. The datasets will then be merged based on geographic identifiers such as county or city and organized into a structured format suitable for analysis. Several machine learning models will be developed and compared, including logistic regression and gradient boosting algorithms. These models will be trained on historical data to identify patterns associated with crime occurrence. Model performance will be evaluated using metrics such as accuracy, precision, recall, and F1-score to determine the most effective approach. To enhance usability and interpretability, an interactive web-based visualization tool will be developed. The domain name for the web application will be purchased through a domain registrar such as Squarespace and connected to the deployed application hosted on Google Cloud Platform. It will display predicted crime hotspots on an interactive map, allowing users to explore patterns spatially. The model’s predictions will also be validated by comparing them to actual crime data not used during training. The results will then be analyzed to assess accuracy and identify any limitations. 

Anticipated Outcomes 

This project is expected to produce a functional machine learning model capable of identifying and predicting criminal hotspots based on historical and socioeconomic data. In addition, the project will result in an interactive web application that visualizes these predictions, allowing users to explore patterns in an intuitive and accessible way. 
Significance 
This project demonstrates how computer science can be applied to address complex real world challenges. The intended audience for this project includes software engineers, data scientists, and researchers in criminology and public policy. This project will provide valuable experience in designing and implementing a machine learning model, from data collection and processing to model development and deployment.
